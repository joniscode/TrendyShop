# TrendyShop

Proyecto de tienda online creado con Vite, JavaScript vanilla, HTML y CSS.

## Tecnologias y librerias

- Vite: servidor de desarrollo y empaquetador del proyecto.
- JavaScript vanilla: componentes, eventos e interacciones del checkout.
- HTML y CSS: estructura, estilos globales y responsive.

Por ahora no se usan librerias externas de UI. Si mas adelante se agrega alguna, debe documentarse en esta seccion.

## Instalacion

```bash
npm install
```

## Ejecutar en desarrollo

```bash
npm run dev
```

Luego abre la URL que muestre la terminal. Normalmente es:

```text
http://127.0.0.1:5173/
```

## Rutas principales

- Inicio: `/`
- Catalogo: `/catalogo/`
- Quienes somos: `/quienes-somos/`
- Contactanos: `/contactanos/`
- Carrito: `/carrito/`
- Checkout: `/checkout/`

## Build de produccion

```bash
npm run build
```

## Vista previa del build

```bash
npm run preview
```
