# 🛍️ TrendyShop - Ecommerce Revival

---

## 📝 Acerca del Proyecto
Este proyecto fue desarrollado durante la **Hackathon de la corte 12 JAVA full stack developer de Generation**. El objetivo principal fue rescatar y transformar digitalmente a **TrendyShop**, una tienda de ropa y accesorios para jóvenes en Colombia fundada en 2018 que entró en crisis en 2025.
